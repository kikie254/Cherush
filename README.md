# Cherush Stay Iten

A boutique hospitality website built with Next.js for a premium guesthouse experience in Iten, Kenya.

## Included

- Public marketing pages for rooms, gallery, experience, about, contact, account, and bookings
- Booking request and contact APIs with seed-data fallback
- Admin dashboard scaffold
- Supabase helper utilities and starter SQL migration
- React Query provider and reusable UI components

## Quick start

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Optional integrations

- Supabase auth and database
- SMTP email notifications
- Google Analytics
- Upstash rate limiting
